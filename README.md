# multitools

ログ調査などで多用するURLエンコードやJSON、その組み合わせ、場合によってはunicodeエンコードなどが含まれるため、１アプリ内で復号して読めるようにすることを目的としたもの。
Macではほとんど類似のものがないため作成した、がモチベーション。
